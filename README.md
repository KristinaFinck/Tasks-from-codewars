# Tasks from Codewars

This repository contains JavaScript exercises and algorithmic tasks
I solved while learning JavaScript fundamentals.

The goal of this repository is to practice core concepts such as:
- loops and nested loops
- working with strings and arrays
- checking uniqueness of elements
- basic sorting algorithms
- understanding time complexity

Some problems are solved in more than one way
to explore different approaches (from basic loop-based solutions
to more modern JavaScript methods).

This repository reflects my learning process and progress over time.
